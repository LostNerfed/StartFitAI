# SynergyFit

SynergyFit es una aplicación minimalista y avanzada de fitness y nutrición optimizada para pantallas AMOLED. 
Su diseño "Liquid Glass" y su interfaz de usuario intuitiva la convierten en la herramienta perfecta para el seguimiento de rutinas de entrenamiento, progresos de levantamiento de peso, y registro nutricional detallado.

## Características Principales

* **Minimalismo AMOLED**: Tema completamente oscuro para ahorro de batería y estética premium con efectos "Glassmorphism".
* **Registro de Entrenamientos**: Sistema avanzado con tracking de series, repeticiones, peso (Lbs/Kg), y PRs (Personal Records). Soporta Dropsets y rutinas libres o guiadas.
* **Diario Nutricional**: Registro detallado de comidas y snacks, calculando automáticamente las calorías y macronutrientes (Proteínas, Carbohidratos, Grasas) en base al consumo diario recomendado.
* **Seguimiento de Progreso**: Gráficos interactivos de evolución de peso y estadísticas de los entrenamientos mensuales.
* **Copias de Seguridad (Backups)**: Exporta e importa fácilmente todas tus rutinas e historial de entrenamiento en un archivo JSON local de forma completamente privada.

## Ejecutar Localmente

**Requisitos Previos:** [Android Studio](https://developer.android.com/studio)

1. Abre Android Studio.
2. Selecciona **Open** y elige el directorio que contiene este proyecto.
3. Permite que Android Studio descargue las dependencias de Gradle.
4. Ejecuta la aplicación en un emulador o dispositivo físico.
